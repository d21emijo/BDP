# Setup
## Python create environment
1. created map for enviornment with: py -m venv .venv
2. activate and use enviornment: .\.venv\Scripts\Activate.ps1

## Python packages
1. install packages: pip install tensorflow pandas numpy scikit-learn matplotlib jupyter
2. save dependancys: pip freeze > requirments.txt

## Python install dependancys
1. read and install same versions: pip install -r requirments.txt
